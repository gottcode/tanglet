How to Play

Using the Mouse

Kelime seçimi: Kelimenin harflerine tıklanılır.

Tahmin yürütme: Son seçilen harfe tıklanılır.

Harfleri silme: Önceki seçilen harfe tıklanılır.

Kelimeyi silme: İlk seçilen harfe çift tıklanılır.

Using the Keyboard

Kelime seçimi: Kelimenin harfi girilir.

Tahmin yürütme: Enter’a basılır.

Kelimeyi silme: Ctrl+Backspace tuşlarına basılır.
