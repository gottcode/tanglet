How to Play

Using the Mouse

Een woord selecteren: Klik op de letters van het woord.

Een woord raden: Klik op de laatste letter van het woord.

Een letter wissen: Klik op een eerder geselecteerde letter.

Een woord wissen: Dubbelklik op de eerste letter van het woord.

Using the Keyboard

Een woord selecteren: Typ de letters van het woord.

Een woord raden: Druk op Enter.

Een woord wissen: Druk op Ctrl+Backspace.
