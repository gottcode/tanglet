How to Play

Using the Mouse

Selectaţi un cuvînt: Faceţi click pe literele unui cuvînt.

Faceţi o presupunere: Faceţi click pe ultima literă selectată.

Ştergeţi literele: Faceţi click pe o literă anterior selectată.

Ştergeţi cuvînt: Faceţi clic de două ori pe prima literă selectată.

Using the Keyboard

Selectaţi un cuvînt: TTastaţi literele cuvîntului.

Faceţi o presupunere: Apăsaţi Enter.

Ştergeţi cuvîntul: Apăsaţi Ctrl+Backspace.
